Set-up Java

1.Download JDK

  1. Download the zip file from this link. Make sure you download the right version for your OS.
  2. Extract the content.

2.Set up JAVA_HOME

  1. Right click on My Computer.
  2. Click on properties.
  3. On the left side click on Advanced system settings.
  4. Click on Environment Variables in the Advanced tab.
  5. Click on the top New button.
  6. At the variable name type JAVA_HOME.
  7. At the variable value enter the extracted jdk folder adress.
  8. Click Ok.
  9. In the bottom tab search for the variable named Path.
  10. Select it and click edit.
  11. In the variable value box add a ";" at the end of the text and then add the jdk bin folder adress.
  12. Check if everything works by using command prompt by typing in javac - version.

Set-up Maven

1. Download Apache Maven

  1. Download the binary zip archive from this link .
  2. Extract the archive's content.

2. Set up Maven variables

  1. Right click on My Computer.
  2. Click on properties.
  3. On the left side click on Advanced system settings.
  4. Click on Environment Variables in the Advanced tab.
  5. Click on the top New button.
  6. At the variable name type MAVEN_HOME.
  7. At the variable value enter the extracted maven folder adress.
  8. Click Ok.
  9. Click on the top New button.
  10. At the variable name type M2_HOME.
  11. At the variable value enter the extracted maven folder adress.
  12. Click Ok.
  13. In the bottom tab search for the variable named Path.
  14. Select it and click edit.
  15. In the variable value box add a ";" at the end of the text and then add the maven bin folder adress.
  16. Check if everything works by using command prompt by typing in mvn - version.

Set-up Postgres

1. Download PostgreSQL.

2. When intall use:

-a simple password ("admin" or "password").

-the default port ("5432").

-the default host ("localhost").

(don't use the stack builder installation)

3. To run the postgres console :

-go to your installation folder and enter the folder "scripts".

-run "runpsql".

-just press "ENTER" until "password for user postgres" shows up , then enter your password.

(there is already a script for you, that will set you up).

-type: \ir (copy and paste the link from eFind-scr-scripts) / (setup or db).sql

(you have to change the \ from the link, to /) and then press ENTER.

*first use the setup and after use db, after running both you are all set :)*
